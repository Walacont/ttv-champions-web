🔧 Admin Debug Tool

Tools zur manuellen Verarbeitung von Doppel-Matches

⚠️ Warnung: Diese Tools sind nur für Admins und Coaches. Verwende sie nur, wenn die automatischen Cloud Functions nicht funktionieren.

📊 Status Check

🔄 Alle unverarbeiteten Doppel-Matches verarbeiten

Diese Funktion findet alle approved Doppel-Matches, die noch nicht verarbeitet wurden, und verarbeitet sie (Elo-Update, Punkte-Historie, Paarungsstatistiken).

🎯 Einzelne Anfrage verarbeiten

Verarbeite eine spezifische Doppel-Anfrage anhand ihrer ID. Die ID findest du in Firestore unter doublesMatchRequests.
